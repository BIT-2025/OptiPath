# OptiPath

OptiPath is a C++ program that models a map using nodes and edges, formulates shortest-path optimization as a linear programming problem using the Big-M method and Simplex algorithm, and visualizes the optimal path with SFML.

## Features

- Shortest path calculation using Linear Programming  
- Implementation of the Big-M method and Simplex algorithm  
- Visual representation of paths with SFML  

## Requirements

- C++ compiler (e.g., GCC)  
- SFML library (version 3.0.2 recommended)
